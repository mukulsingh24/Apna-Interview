# AI Resume Analyzer & Interview Preparation Platform

An AI-powered web application designed to help users optimize resumes, evaluate ATS compatibility, identify missing skills, and prepare for technical interviews through personalized question generation and feedback.

---

## 🚀 Features

* Resume Upload & Parsing
* ATS Score Evaluation
* Skill Gap Detection
* AI-Based Resume Suggestions
* Role-Based Interview Question Generation
* Personalized Interview Preparation Dashboard
* Secure User Authentication
* Responsive Modern UI
* Real-Time Resume Analysis

---

## 🛠️ Tech Stack

### Frontend

* Next.js
* TypeScript
* Tailwind CSS

### Backend

* Node.js
* Express.js
* REST APIs

### Database & Authentication

* PostgreSQL
* Firebase Authentication

### AI Integration

* Gemini API / OpenAI API

---

## 📌 Project Objective

The goal of this platform is to simplify the resume screening and interview preparation process by leveraging AI-powered analysis and automation. The platform helps students and job seekers improve resume quality, identify missing technical skills, and prepare for interviews more effectively.

---

## ⚙️ Core Modules

### 1. Resume Analyzer

* Extracts resume content
* Evaluates ATS compatibility
* Detects missing skills
* Suggests resume improvements

### 2. Interview Preparation

* Generates role-specific interview questions
* Provides technical and HR interview practice
* Tracks preparation progress

### 3. Authentication System

* Secure login/signup using Firebase Authentication
* Protected user dashboard

### 4. Dashboard

* Resume upload history
* Analysis reports
* Saved interview questions
* Personalized recommendations

## 💻 Installation & Setup

### Clone Repository

```bash
git clone https://github.com/your-username/AI-Resume-Analyzer.git
```

### Install Dependencies

```bash
npm install
```

### Start Frontend

```bash
npm run dev
```

### Start Backend

```bash
npm start
```
